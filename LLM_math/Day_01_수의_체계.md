## 📖 핵심 개념

### 1. 수의 진화: 문제를 해결하기 위한 여정

#### 1.1 자연수 (Natural Numbers)
```
ℕ = {1, 2, 3, 4, 5, ...}
```

**등장 배경**: 사물을 세기 위해
- "사과가 3개 있다"
- "양이 10마리 있다"

**한계**: 빼기를 하면 문제가 생긴다
- 3 - 5 = ? (자연수로 표현 불가)

---

#### 1.2 정수 (Integers)
```
ℤ = {..., -3, -2, -1, 0, 1, 2, 3, ...}
```

**등장 배경**: 빼기와 빚을 표현하기 위해
- "5에서 3을 빼면 2"
- "3에서 5를 빼면 -2" → 음수 등장!

**해결된 문제**:
- 빼기 연산이 항상 가능해짐
- 방향(앞/뒤, 위/아래)을 표현 가능

**한계**: 나누기를 하면 문제가 생긴다
- 3 ÷ 2 = ? (정수로 표현 불가)

---

#### 1.3 유리수 (Rational Numbers)
```
ℚ = {a/b | a, b ∈ ℤ, b ≠ 0}
```

**등장 배경**: 나누기와 비율을 표현하기 위해
- 3 ÷ 2 = 3/2 = 1.5
- "피자 하나를 3명이 나눠 먹으면 각자 1/3씩"

**해결된 문제**:
- 나누기 연산이 (0으로 나누는 것 제외하고) 가능해짐
- 비율, 확률, 비례 표현 가능

**한계**: 제곱근을 구하면 문제가 생긴다
- √2 = ? (유리수로 표현 불가)

---

#### 1.4 실수 (Real Numbers)
```
ℝ = 유리수 + 무리수
```

**등장 배경**: 연속적인 양과 제곱근을 표현하기 위해
- √2 = 1.41421356...
- π = 3.14159265...
- e = 2.71828182...

**해결된 문제**:
- 모든 제곱근 표현 가능
- 연속적인 값 (길이, 시간, 온도 등) 표현 가능
- 미적분학의 기초

**한계**: 제곱해서 -1이 되는 수는?
- x² = -1을 만족하는 실수는 없음

---

#### 1.5 복소수 (Complex Numbers)
```
ℂ = {a + bi | a, b ∈ ℝ, i² = -1}
```

**등장 배경**: x² + 1 = 0 같은 방정식을 풀기 위해
- i² = -1 정의
- 3 + 4i, -2 + 7i 등

**해결된 문제**:
- 모든 다항방정식이 해를 가짐 (대수학의 기본정리)
- 신호 처리, 양자역학에서 필수

---

### 2. 스칼라 vs 벡터

#### 2.1 스칼라 (Scalar)
**정의**: 크기만 가진 수
- 예: 온도 23℃, 속력 60km/h, 질량 5kg

**특징**:
- 단 하나의 숫자로 표현
- 방향이 없음

---

#### 2.2 벡터 (Vector)
**정의**: 크기와 방향을 동시에 가진 수

**2차원 벡터**:
```
v = [3, 4]
```
- 첫 번째 숫자: x 방향으로 3
- 두 번째 숫자: y 방향으로 4

**예시**:
- 속도: [30, 40] km/h (동쪽으로 30, 북쪽으로 40)
- 위치: [x, y]
- RGB 색상: [255, 128, 0] (빨강, 초록, 파랑)

**LLM에서의 벡터**:
- 단어 임베딩: "사과" → [0.2, -0.5, 0.8, ..., 0.1] (512차원)
- 문장 임베딩: "안녕하세요" → [0.1, 0.3, -0.2, ..., 0.5]

**왜 벡터인가?**
- 단어의 의미를 숫자 하나로 표현할 수 없음
- 여러 차원에서 의미를 표현 (동의어, 반의어, 문맥 등)

## 💻 Python 실습

### 실습 1: 수의 체계 탐색
```python
import numpy as np

# 1. 자연수
natural = 5
print(f"자연수: {natural}")

# 2. 정수 (음수 포함)
integer = -3
print(f"정수: {integer}")

# 3. 유리수 (분수)
rational = 3 / 4
print(f"유리수: {rational}")

# 4. 무리수
import math
irrational = math.sqrt(2)
print(f"무리수 √2: {irrational}")

# 5. 복소수
complex_num = 3 + 4j
print(f"복소수: {complex_num}")
print(f"복소수의 실수부: {complex_num.real}")
print(f"복소수의 허수부: {complex_num.imag}")
```

### 실습 2: 스칼라 vs 벡터
```python
import numpy as np

# 스칼라: 단일 값
temperature = 23  # 온도
speed = 60        # 속력

print(f"온도 (스칼라): {temperature}℃")
print(f"속력 (스칼라): {speed} km/h")

# 벡터: 여러 값의 배열
position = np.array([3, 4])  # 2D 위치
velocity = np.array([30, 40])  # 2D 속도

print(f"\n위치 (벡터): {position}")
print(f"속도 (벡터): {velocity}")

# 벡터의 크기 계산
magnitude = np.linalg.norm(position)
print(f"\n위치 벡터의 크기: {magnitude}")

# LLM 스타일: 고차원 벡터
word_embedding = np.random.randn(512)  # 512차원 벡터
print(f"\n단어 임베딩 차원: {word_embedding.shape}")
print(f"단어 임베딩 일부: {word_embedding[:5]}")
```

### 실습 3: 벡터로 색상 표현
```python
import numpy as np
import matplotlib.pyplot as plt

# RGB 색상을 벡터로 표현
red = np.array([255, 0, 0])
green = np.array([0, 255, 0])
blue = np.array([0, 0, 255])
purple = np.array([128, 0, 128])

# 색상 시각화
colors = np.array([red, green, blue, purple]) / 255.0
color_names = ['Red', 'Green', 'Blue', 'Purple']

fig, axes = plt.subplots(1, 4, figsize=(12, 3))
for ax, color, name in zip(axes, colors, color_names):
    ax.add_patch(plt.Rectangle((0, 0), 1, 1, color=color))
    ax.set_xlim(0, 1)
    ax.set_ylim(0, 1)
    ax.set_title(f'{name}\n{color}')
    ax.axis('off')

plt.tight_layout()
plt.savefig('color_vectors.png', dpi=150, bbox_inches='tight')
print("색상 벡터 이미지 저장 완료!")
```

## 🔗 LLM 연결점

### 1. 왜 LLM은 벡터를 사용하는가?

**문제**: 컴퓨터는 숫자만 이해할 수 있음
- "사과" → ? (컴퓨터가 이해할 수 있는 형태로 변환 필요)

**해결책**: 단어를 벡터로 변환 (Word Embedding)
```
"사과" → [0.2, -0.5, 0.8, 0.3, ...]  (512차원)
"배"   → [0.3, -0.4, 0.7, 0.2, ...]
```

**벡터의 장점**:
1. 의미의 유사도 계산 가능
   - "사과"와 "배"의 벡터는 가까움 (둘 다 과일)
   - "사과"와 "자동차"의 벡터는 멈

2. 의미 연산 가능
   - "왕" - "남자" + "여자" ≈ "여왕"

3. 고차원 공간에서 복잡한 관계 표현
   - 512차원 공간에서 단어들의 관계 학습

---

### 2. 임베딩의 본질
**임베딩 = 단어/문장을 고차원 벡터로 변환**

예시:
```python
# "안녕하세요"를 512차원 벡터로
embedding = model.encode("안녕하세요")
# → [0.12, -0.34, 0.56, ..., 0.78]  (512개 숫자)
```

이 벡터의 각 차원은:
- 의미의 특정 측면을 표현 (추상적)
- 인사, 긍정/부정, 격식체/반말 등
