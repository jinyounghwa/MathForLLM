## 📖 핵심 개념

### 1. 확률의 정의
```
P(A) = (A가 일어나는 경우의 수) / (전체 경우의 수)

0 ≤ P(A) ≤ 1

P(A) = 0: 불가능
P(A) = 1: 확실
P(A) = 0.5: 50-50
```

### 2. 확률의 기본 규칙
```
덧셈 규칙 (OR):
P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

곱셈 규칙 (AND):
P(A ∩ B) = P(A) × P(B|A)
```

### 3. 조건부 확률
```
P(A|B) = P(A ∩ B) / P(B)

"B가 일어났을 때 A가 일어날 확률"
```

### 4. 독립
```
A와 B가 독립:
P(A ∩ B) = P(A) × P(B)
P(A|B) = P(A)
```

## 💻 Python 실습

```python
import numpy as np

# 주사위 시뮬레이션
n_rolls = 10000
rolls = np.random.randint(1, 7, n_rolls)

print("=== 주사위 확률 ===\n")

for i in range(1, 7):
    prob = np.sum(rolls == i) / n_rolls
    print(f"P({i}) = {prob:.4f} (이론: 0.1667)")

# 조건부 확률: P(홀수|3 이상)
odd = (rolls % 2 == 1)
ge3 = (rolls >= 3)

p_odd_given_ge3 = np.sum(odd & ge3) / np.sum(ge3)
print(f"\nP(홀수|3 이상) = {p_odd_given_ge3:.4f}")
print(f"이론값: 2/4 = 0.5")
```
